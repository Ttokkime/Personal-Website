This is a code I made in the past during my freshman year of highschool using javascript language.

I made a code that simulated the Hailstone Series concept.

The code first starts off with an integer value that is greater than 0, which in this case will be x.

x is even, then the next value in the series is x/2. If x is odd, then the next value in the series is 3x + 1.

The series will inevitably end with the value 1 at the very end.
